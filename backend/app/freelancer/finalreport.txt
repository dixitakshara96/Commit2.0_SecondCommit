7. Manual Setup Required Before Deployment
1. Start PostgreSQL — Ensure PostgreSQL is running on the configured port (default: 5433)
2. Apply migrations —  cd backend && alembic upgrade head 
3. Run seed script —  cd backend && PYTHONPATH=. python seed.py 
4. Configure  .env  — Set:
-  DATABASE_URL  — PostgreSQL connection string
-  SECRET_KEY  — Random long string for JWT signing
-  GITHUB_TOKEN  — GitHub personal access token (for API calls)
-  OLLAMA_BASE_URL  — Ollama server URL (default:  http://localhost:11434 )
-  OLLAMA_MODEL  — Model name (default:  gemma3:4b )
5. Start Ollama — Ensure Ollama is running and the configured model is pulled
6. Start backend —  cd backend && uvicorn app.main:app --reload 
7. Start frontend —  cd frontend && npm run dev
